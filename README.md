# 📄 AI Research Paper Summarizer

A simple web app that summarizes research papers (PDFs) into clear and structured summaries using AI.

---

## 🚀 Features

* Upload PDF files
* Get summarized output instantly
* Extracts key ideas, methods, and results
* Handles large documents using chunking

---

## 🛠️ Tech Used

* Python (Flask)
* HTML, CSS, JavaScript
* Groq API (LLaMA 3.1)
* LangChain
* pdfplumber

---

## 📁 Project Structure

app.py
utils.py
templates/index.html
static/css/style.css
static/js/script.js

---

## ⚙️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Create `.env` file:
   GROQ_API_KEY=your_api_key_here

3. Run:
   python app.py

Open: http://127.0.0.1:5000

---

## ⚠️ Important

* Do NOT upload `.env` file
* Requires API key to run
